Next.js 使用基於檔案的路由系統

一個名為 `page.js` 的檔案名稱簡單地告訴 NextJS 渲染一個頁面

創建一個包含 `page.js` 的新資料夾可以新增一個新路由,路徑名稱等於資料夾名稱

Next 和 React 在靜態路由和動態路由方面的差異:

**Next**

-   靜態路由: 直接透過檔案系統創建
-   動態路由: 使用方括號 `[]` 創建動態段

**React**

-   靜態路由: 通常使用第三方函式庫如 React Router 手動定義
-   動態路由: 需要在路由設定中明確指定參數
